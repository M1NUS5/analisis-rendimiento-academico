Análisis del Rendimiento Académico
Este proyecto analiza el impacto de los hábitos de estudio en el rendimiento académico utilizando herramientas de Business Intelligence.

Contenido del repositorio
- Dataset en Excel
- Dashboard en Power BI
- Reporte en PDF

Objetivo

Analizar la relación entre:

- Horas de estudio
- Asistencia
- Tareas entregadas
- Calificación final

Herramientas utilizadas
- Power BI
- Excel
- GitHub

Resultados

Se encontró una relación positiva entre las horas de estudio y el rendimiento académico, así como la influencia de la asistencia y la constancia.
